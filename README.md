# 🚀 Customer Churn Prediction

This project predicts whether a customer will churn based on their usage and service details.

## 🔥 Features
- Machine Learning pipeline using Scikit-learn
- Real-time prediction using Streamlit
- Probability score for churn risk
- User-friendly web interface

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## 📊 Inputs
- Tenure
- Monthly Charges
- Contract Type
- Internet Service
- Payment Method

## 🎯 Output
- Churn Prediction (Yes/No)
- Probability Score
