# 🚀 Customer Churn Prediction Web App

A Machine Learning-based web application that predicts whether a customer is likely to churn based on their service usage and account details.

---

## 🔥 Features

* End-to-end Machine Learning pipeline using Scikit-learn
* Real-time churn prediction through an interactive web app
* Displays churn probability for better decision-making
* Clean and user-friendly interface built with Streamlit

---

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn
* Streamlit

---

## 📊 Input Parameters

The model uses the following customer details:

* Tenure (in months)
* Monthly Charges
* Contract Type
* Internet Service
* Payment Method

---

## 🎯 Output

* Churn Prediction (Yes / No)
* Probability Score indicating churn risk

## 🧠 How It Works

* Data is preprocessed using a pipeline (scaling + encoding)
* A machine learning model is trained on customer data

---

## 💡 Use Case

This project helps businesses:

* Identify customers at risk of leaving
* Take preventive actions (offers, retention strategies)
* Improve customer retention and revenue
